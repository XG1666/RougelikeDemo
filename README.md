# RougelikeDemo
闲来无事做的一个弱智小游戏 
  
# 背景  
你是一位奉旨讨伐恶灵的勇士，你需要在迷宫中不断击败小怪，强化自身，最终在关底击败恶灵  

# 如何开玩  
只需三步！  
- **下载.zip文件**
- **解压文件**
- **点击.exe文件即可开玩**
